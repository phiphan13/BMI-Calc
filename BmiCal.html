BMI Guide

Under Weight = Less than 18.6

Normal Range = 18.6 - 24.9

Overweight = 25 - 29,9

Pre-Obese = 30.0 - 35.0

Obesity Class I = 30.0 - 35.0

Obesity Class II = 35.0 - 40.0

Obesity Class III = 40+

NOTE: The BMI now is very inaccurate if you are physically active due to not considering muscle mass.
